## 🐛 故障排除

### 问题1: 看不到星期几，仍然显示完整时间

**原因**: 浏览器缓存未更新

**解决方案**:
1. **强制刷新页面**:
   - Chrome/Edge: `Ctrl + Shift + R` (Windows) 或 `Cmd + Shift + R` (Mac)
   - Firefox: `Ctrl + F5` (Windows) 或 `Cmd + Shift + R` (Mac)

2. **清除浏览器缓存**:
   - Chrome: 设置 → 隐私和安全 → 清除浏览数据
   - Edge: 设置 → 隐私、搜索和服务 → 清除浏览数据

3. **使用无痕模式**:
   - Chrome/Edge: `Ctrl + Shift + N`
   - Firefox: `Ctrl + Shift + P`

### 问题2: 星期几显示错误

**验证方法**: 使用在线日历工具验证
```
https://www.calendar-365.com/
```

**示例**:
- 2023年12月30日 → 应该是 **星期六** (六)
- 2023年12月28日 → 应该是 **星期四** (四)

### 问题3: 其他彩种也显示星期几

**原因**: 代码逻辑错误

**检查**: 查看浏览器控制台是否有JavaScript错误
```
F12 → Console 标签
```

**预期**: 只有 `lotCode = '60001'` (香港六合彩) 才显示星期几
